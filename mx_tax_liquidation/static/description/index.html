¿Qué es MX Tax Liquidation?

MX Tax Liquidation es un módulo para Odoo 17 Enterprise diseñado para liquidar las obligaciones fiscales mensuales de empresas mexicanas (ISR propio, IVA por pagar, retenciones a terceros) directamente desde los saldos contables registrados en el sistema.

"El módulo es un liquidador de saldos contables. No recalcula impuestos, no recorre facturas. Solo cancela los pasivos fiscales ya registrados contra bancos."

Flujo del proceso

1. Borrador → 2. Calcular saldos → 3. Confirmar → 4. Pagar (parcial / total) → 5. Pagado ✓

Estados de la liquidación

Borrador Confirmado Pago Parcial Pagado Cancelado

Características principales

📊

Lectura SQL optimizada

Los saldos se calculan con una sola consulta SQL parametrizada sobre account_move_line, sin bucles ORM ni carga de registros en memoria.

⚛️

Reclasificación atómica IVA

El IVA retenido genera un asiento de dos líneas: DEBE cuenta_fuente → HABER banco. Sin pasos intermedios, sin duplicar la obligación.

💸

Pagos parciales duales

Soporta distribución automática (prorrateo proporcional con ajuste de centavos) y manual (monto exacto por concepto).

🏢

Multicompañía nativo

Reglas de seguridad ir.rule garantizan aislamiento total. Cada empresa opera en su propio espacio de datos.

🔒

Pista de auditoría inmutable

Cada acción relevante queda registrada en mx.tax.settlement.log. Los registros de auditoría no se pueden modificar ni eliminar.

⚡

Control de concurrencia

SELECT FOR UPDATE NOWAIT en confirmación y pago previene doble procesamiento en entornos multiusuario.

📋

Reportes QWeb PDF

Reporte de liquidación detallado y reporte de conciliación con semáforo visual (✅ ⚠️ 🔴) listos para firma y archivo.

🛡️

4 niveles de seguridad

Grupos: Usuario, Gerente, Auditor y Configuración con permisos granulares por modelo.

Instalación rápida

  1. Copiar la carpeta mx_tax_liquidation al directorio de addons.
  2. Reiniciar el servidor Odoo.
  3. Actualizar la lista de módulos desde Ajustes → Técnico → Actualizar lista de módulos.
  4. Instalar MX Tax Liquidation desde la lista de aplicaciones.
  5. Ir a Contabilidad → Configuración → Conceptos Fiscales Liquidación y mapear las cuentas de cada concepto.
